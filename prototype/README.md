# OrbitQuest frontend concept

Статический интерактивный прототип без сборки и зависимостей.

## Запуск

Откройте `index.html` двойным кликом или запустите любой статический HTTP-сервер.

## Реализовано

- responsive desktop/mobile layout;
- переключение Today / Skill Map / Radar;
- прогресс ежедневной миссии;
- интерактивные ответы;
- dialog добавления нового трека;
- панель AI-наставника;
- состояния skill nodes.
