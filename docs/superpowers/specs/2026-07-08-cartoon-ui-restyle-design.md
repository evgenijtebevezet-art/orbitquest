# OrbitQuest — рестайл UI под утверждённый мульт-стиль (Рик и Морти)

## Контекст

Арт (пролог, стайлборд №6) утверждён и выдержан: взрослый 2D-мульт — жирные контуры,
тёмный захламлённый корабль, кислотные акценты, глифы. UI-обвязка осталась временной
(«стеклянный неон») и с артом не рифмуется. Владелец выбрал **полный мульт-стиль**.

## Решения (утверждены в чате)

1. **Палитра из арта.** Токены `:root`:
   - фон `--bg: #141028` (сине-фиолетовый космос), панель `--panel: #251b40`,
     панель-глубже `--panel-deep: #1b1332`, «чернила» контуров `--ink: #0d0a1c`;
   - акценты: портальный зелёный `--acid: #8ef65b` (главные CTA), циан `--cyan: #52e5ff` (KORA),
     маджента `--magenta: #ff5fd7`, янтарный `--amber: #ffc247` (PIX), тревога `--danger: #ff4f6d`,
     приглушённый `--muted: #8f86ad`;
   - семантика персонажей сохраняется: KORA=циан, PIX=янтарь, VEGA/успех=зелёный.
2. **Мульт-приёмы.** Панели/кнопки: контур `3px solid var(--ink)` + жёсткая офсетная тень
   `4px 4px 0 var(--ink)`; нажатие кнопки = translate(4px,4px) + тень 0 («вдавливание»).
   Панели — скошенный верхний правый угол (clip-path) и ряд CSS-«заклёпок» (::before, radial-gradient
   dots) на крупных карточках. Мягкий глоу остаётся только на акцентах в духе арта (ядро KORA,
   активные индикаторы). Код-блоки — «бортовой терминал»: тёмный экран с рамкой-бизелем.
3. **Шрифты (самохостинг через @fontsource, попадают в прекеш PWA):**
   - заголовки/дисплей: **Unbounded** (600, 800), кириллица+латиница;
   - текст/кнопки: **Rubik** (400, 500, 700), кириллица+латиница;
   - код: **JetBrains Mono** (400), кириллица+латиница.
   Импорт только нужных сабсетов (`cyrillic-*.css`, `latin-*.css`).
4. **Скоуп.** Только визуальный слой: полная переписка `apps/web/src/styles.css` (заодно
   удаляется мёртвый CSS старого прототипа: space-window, satellite, vega-signal, kora-orb,
   pix-unit, mission-console, connection-pill и пр. — эти классы не рендерятся с T13),
   импорты шрифтов в `main.tsx`, `theme-color` в `index.html`, цвета в `manifest.webmanifest`.
   Логика, разметка, тексты компонентов — без изменений.
5. **Приёмка.** `typecheck + build + smoke` зелёные; e2e-скрипт приёмки — 29/29 на 390×844
   и 360×800 (селекторы текстовые, разметка не меняется); скриншоты ключевых экранов
   (пролог, калибровка, Мостик, задание, Atlas) владельцу на утверждение.

## Вне скоупа

Иллюстрированные фоны палуб из арта, анимации персонажей, перегенерация арта, звук.
