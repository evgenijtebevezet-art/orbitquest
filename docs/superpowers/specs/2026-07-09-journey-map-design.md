# OrbitQuest — «Путешествие Odyssey»: карта-маршрут, аварии, 5 заданий, живое общение

Фидбек игрока №1: «ориентация непонятна, куда какой код», «4 задания мало — давай 5»,
«проработаем общение», «хочу путешествие как в Duolingo: Земля → МКС → Луна, и экстренные
починки корабля по пути». Утверждено: **маршрут + ветка**.

## Структура путешествия

Одна вертикальная тропа (Duolingo-модель: активен ровно один следующий узел, остальные
с замками; пройденные — отмечены; пропущенные калибровкой — «уже знаешь», визуально пройдены).

- **Этап 1 «Земля → МКС»:** CODE-01 → CODE-02 → ⚠SOS-01 → CODE-03
- **Этап 2 «МКС → Луна»:** CODE-04 → ⚠SOS-02 → CODE-05 → CODE-06
- **Ветка на Луне «Бортовой ИИ» (Луна → Марс):** AGENT-01 → ⚠SOS-03 → AGENT-02 → AGENT-03
  (открывается по достижении Луны; развилка видна на карте)

Выбор «курса» из калибровки удаляется: reveal показывает карту навыков + применённые
пропуски + кнопку «В путь» (sector в профиле проставляется автоматически, схема не меняется).
Переключатель курса убирается; палубы: «Путешествие» (главная) / «Карта навыков» / «Профиль».

## Аварийные миссии (новый контент, contextTag="new" → путь к transfers)

Код аварий = системы корабля (отвечает на «куда какой код»):
- **SOS-01 «Кислородный контроллер»** (этап 1): скрипт вентиляции; закрепляет
  literal/variables/order в новом контексте. skillId: code.variables.
- **SOS-02 «Автопилот стыковки с Луной»** (этап 2): условия/порядок в коде автопилота.
  skillId: code.conditions.
- **SOS-03 «Бортовой ИИ чудит»** (ветка): ассистент корабля галлюцинирует конфиг;
  проверка AI-кода. skillId: agent.verification.

Аварии — фиксированные узлы тропы (детерминированно, без рандома), с иконкой ⚠,
своим брифингом «ТРЕВОГА» (PIX ведёт, KORA поддерживает) и рамкой alert в плеере.

## Контент-апгрейд

- Все 9 существующих миссий расширяются до **5 заданий** (сейчас 3–4); новые задания —
  те же 5 типов, сложность нарастает внутри миссии, proof остаётся последним.
- 3 SOS-миссии × 5 заданий. Итого 12 миссий × 5 = 60 заданий.
- **Живое общение:** новое опциональное поле задания `interlude` —
  1–2 реплики экипажа после решения задания (перепалки KORA↔PIX, комментарии VEGA),
  показываются на экране разбора с портретами. Схема/валидатор дополняются
  (опционально, обратная совместимость).

## Карта (арт генерируется codex-image)

Вертикальное полотно (portrait, стиль №6): Земля внизу → орбита с МКС → Луна →
развилка → Марс вверху; пунктирная трасса, тёмный космос, кислотные акценты.
Узлы — HTML-оверлей поверх арта, координаты (x,y в %) в `content/journey.json`:
`{ stages: [{id, title, from, to}], nodes: [{missionId, stage, x, y, kind: "mission"|"emergency"}] }`
+ валидатор (missionId существуют, координаты 0..100, порядок = порядок прохождения).
Маркер корабля Odyssey стоит у последнего пройденного узла (Motion-перелёт при прогрессе).

## UI

- Экран «Путешествие» = прокручиваемая карта (старт снизу), активный узел пульсирует,
  тап по активному → брифинг; по пройденному → повтор; по замку → подсказка «сначала …».
  Карточка «Миссия на сегодня» закрепляется поверх низа карты.
- Плеер миссии не меняется, кроме: interlude-реплики на экране разбора; alert-стиль SOS.
- «Карта навыков» (Atlas) остаётся как есть (вторая палуба).
- Прогрессивное раскрытие сохраняется: до первой пройденной — только карта путешествия.
