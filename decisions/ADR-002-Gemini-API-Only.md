# ADR-002: Только Gemini API в runtime MVP

## Контекст

У проекта есть доступ к Gemini Developer API Free Tier. Использовать локальные CLI пользователь не хочет.

## Решение

Runtime MVP вызывает только Gemini Developer API через серверный AI Gateway. Codex CLI, Claude Code и Gemini CLI не являются зависимостями приложения. В мире продукта Gemini не называется отдельным персонажем: пользователь взаимодействует с KORA, а provider остаётся сменной инфраструктурой.

## Причины

- приложение работает независимо от установленного ПО;
- единый способ авторизации;
- контролируемые JSON-контракты;
- простое развёртывание;
- пригодность для web/mobile клиента;
- возможность ограничить и измерить использование.

## Ограничения

- квота Free Tier не подходит для неконтролируемого публичного масштаба;
- бесплатный tier требует осторожности с данными;
- деградация при исчерпании лимита обязательна;
- доступность preview-моделей не гарантируется.

## Пересмотр

Решение пересматривается после подтверждения retention и появления реальной нагрузки, а не заранее.
