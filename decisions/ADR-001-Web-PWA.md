# ADR-001: Web/PWA вместо игрового движка

## Контекст

Продукт использует игровую механику, карту навыков и анимации, но его основное взаимодействие — текст, карточки, код и граф данных.

## Решение

Начать с responsive web-приложения с возможностью установки как PWA.

## Причины

- единая версия для desktop и mobile;
- быстрые обновления;
- нормальная работа с текстом и кодом;
- доступность и клавиатурная навигация;
- более дешёвый MVP;
- отсутствие тяжёлого игрового runtime.

## Последствия

Сложные 3D-механики не входят в ранние версии. Игровое ощущение создаётся картой, состояниями, анимациями, прогрессом и миссиями.
